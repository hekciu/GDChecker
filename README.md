# @GDChecker

- App used to partially decompile ethereum contracts in order to fetch liquidity method ID and antibot measures.

## Required

- Node.js 16+

## How to using

- Lunch commands
  - `npm start`
    - run cli
  - `npm run dev-ts`
    - run dev cli
  - `npm run type-check`
    - run type-check
  - `npm run build`
    - run swc
      - this only development, not production
  - `npm run test`
    - launches test runner by watch mode
  - `npm run coverage`
    - ~~get coverage report~~
  - `npm run lint`
    - lints code
